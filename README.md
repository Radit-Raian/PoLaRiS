# PoLaRiS: Point-spread Light Reconstruction of Stellar Sources

PoLaRiS (Point-spread Light Reconstruction of Stellar Sources) is an automated pipeline for point spread function (PSF) modeling and reconstruction from multi-band astronomical imaging data. It is currently optimized for the Kilo-Degree Survey and the DESI Legacy Imaging Surveys.

## Step 0: Getting Started

This step sets up all required dependencies for PSF modeling, including both Python libraries and SExtractor. 

### 0.1 Clone the repository

```bash
git clone https://github.com/Radit-Raian/PoLaRiS.git
cd PoLaRiS
pip install -r requirements.txt
```
### 0.2 Building SExtractor

```bash
git clone https://github.com/astromatic/sextractor
cd sextractor
sh autogen.sh
./configure
make -j
sudo make install
sex
```

## Step 5: Star Selection

- Apply selection criteria using CLASS_STAR > 0.8 and classprob_dsc_combmod_star > 0.98 to identify high-confidence stars
- Overlay confirmed stars on the imaging data for visual validation
- Generate the Magnitude vs. Flux Radius plot
- Define patches to divide the PSF into core, intermediate, and optionally two outer regions.

  
---



## Step 6: Cutout Generation

- Generate image cutouts based on the previously defined patches
- Filter out contaminated edgy cutouts by applying a data-quality threshold, retaining only those with at least 40% usable data coverage


---



## Step 7: Masking

- Generate segmentation cutouts from the previously created image cutouts
- Apply primary masking by setting target stars and background regions to 0, and all remaining pixels to 1
- Create a final mask by inverting the previous logic
- Produce images by combining the final mask with the original cutouts through element-wise multiplication
- Apply `binary dilation` to account for extended wings and to improve the contamination removal


---



## Step 8: Stacking

- Normalise the PSF before stacking
- Compute mean stack to characterize the average PSF structure
- Generate median stack for our final analysis across all patches


---



## Step 9: Radial Profile Construction

- Generate radial surface brightness profiles from the median-stacked PSF images
- Normalize and scale individual profiles by proper scaling factors
- Combine the profiles from all four PSF patches by stitching them into a single continuous radial profile
